COLOCANDO FONTES

SAFE COMBINATIONS

O Visual Studio Code nos oferece alguns tipos de fonte, como a Arial, Times New Roman e a Verdana, que são fontes bastante comuns. Podemos usá-las em nossos documentos determinando a propriedade font-family: 

   body {
   font-family: Arial;
}

Normalmente colocamos mais de uma fonte, para caso a que queremos no nosso site não tenha suporte no navegador de quem está acessando: 

   body {
   font-family: Arial, Helvetica, sans-serif;
}

Essa combinação é um exemplo de safe combination e aparece como opção quando colocamos a font-family.

USANDO GOOGLE FONTS

Como já falamos, o Visual Studio Code nos fornece uma quantidade de fontes muito limitada. Por isso, se quisermos usar uma fonte que não aparece nas opções, um bom site para consegui-la é o Google Fonts.

1 Passo: Escolher uma fonte

2 Passo: Selecionar o(s) estilos(s)

3 Passo: Copiar o código import

4 Passo: Colar o código na tag style 

<style>
 @import url('...')
</style>

A partir daí será possível utilizar a fonte em qualquer lugar de seu site, determinando a font-family para o nome da fonte, que, nesse exemplo, é Kanit.

COLOCANDO FONTES BAIXADAS

Apesar do Google Fonts ter uma enorme variedade de fontes, podemos encontrar uma fonte que queiramos usar e que ele não tenha. Por isso, precisaremos baixar em um outro site. Um bom site para fazer isso é o Dafont.

1 Passo: Escolher a fonte

2 Passo: Baixá-la clicando no botão "Baixar"

3 Passo: Abrir o arquivo baixado na pasta

4 Passo: Clicar no arquivo e em "Extrair tudo"

5 Passo: Depois de extrair tudo aparecerá um ou mais arquivos, arraste-os para o arquivo do documento.

6 Passo: Crie o código @font-face na tag style do documento:

   <style> 
   @font-face {
   font-family: 'nome da fonte' ;
   src: url('nome do arquivo') format('formato do arquivo');
}
</style>
A propriedade font-family contém o nome que se refere a fonte que você baixou, portanto pode ter o nome que você quiser e, a partir daí, sempre que for mencionar essa fonte para usá-la, precisará usar esse mesmo nome.

No format você coloca o formato da fonte. Se for ttf, seu formato é truetype e se for otf, seu formato é opentype. E, claro, tem como colocar mais de um formato, caso o navegador não tenha compatibilidade com algum deles.
