PROPRIEDADES DE FONTE

FONT-SIZE

Existem várias unidades usadas para medir fonte, como pt, px, pc, cm, em, rem, in, entre outras. Mas as recomendadas pela W3C para a reprodução na tela são duas: px e em.

Px significa pixel e o tamanho normal de uma fonte é geralmente 16px

A unidade de medida em é relativa à altura do M maiúsculo de uma fonte.

1 em = 16px

A propriedade usada para definir o tamanho é font-size:

    <style> 
body {
    font-size: 2em;  
 }
 </style>
 FONT-WEIGHT

A propriedade font-weight se refere ao peso(grossura) da fonte. Os principais tipos são:

Lighter - 300

Normal - 400

Bold - 700

Bolder - 800

Lembrando que nem todas as fontes possuem esses 4 tipos de peso.

FONT-STYLE

Outra propriedade de fonte é a font-style, que é usada para deixar a fonte em itálico usando o valor italic: 

   <style> 
body {
    font-style: italic;  
}
</style>
 TEXT-DECORATION

E, por último, existe a propriedade text-decoration, que é usada para deixar o texto sublinhado:

   <style> 
body {
    text-decoration: underline;  
}
</style>
 SHORTHAND FONT

Como ficar criando um monte de propriedades de fonte não é tão prático, existe uma shorthand que pode nos ajudar a encurtar o código:

   <style> 
body {
    font: italic bold 1.5em Arial;  
}
</style>
 Existe uma ordem que deve ser seguida dentro da shorthand font:

font-style

font-weight

font-size

font-family

 TEXT-ALIGN

Usamos a propriedade text-align para configurar o tipo de alinhamento do nosso texto:

    <style> 
body {
    text-align:" ";  
 }
 </style>
 Podemos colocar dentro dessa propriedade os seguintes valores:

TEXT-INDENT

Podemos também configurar o espaço no começo de cada parágrafo com a propriedade text-indent:

   <style> 
p {
    text-indent: 50px;  
}
</style>
