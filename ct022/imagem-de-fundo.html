IMAGEM DE FUNDO

COLOCANDO IMAGENS DE FUNDO

Assim como para o efeito degradê, usaremos a propriedade background-image para colocarmos uma imagem de fundo dentro de uma caixa:
style
body {
background-image: url(imagem.png);
}
style

Quando testamos no navegador, podemos perceber que ele irá automaticamente replicá-la quantas vezes for necessário para ocupar todo o espaço da tela.

BACKGROUND-REPEAT

Podemos mudar a propriedade background-repeat se quisermos que a imagem não se repita ou ajustar a sua repetição:

style
body {
background-image: url(imagem.png);
background-repeat: no-repeat;
}
style

Quando ajustamos para no-repeat a imagem simplesmente não se repete

style
body {
background-image: url(imagem.png);
background-repeat: repeat-x;
}
style

A propriedade repeat-x faz a imagem se repetir somente no eixo x.

style
body {
background-image: url(imagem.png);
background-repeat: repeat-y;
}
style

A propriedade repeat-y faz a imagem se repetir somente no eixo y.

BACKGROUND-SIZE

Podemos também redimensionar a imagem usando a propriedade background-size:

style
body {
background-size: 10px 20px;
}
style

Fazendo essa configuração, a imagem passará a ter 10px de largura e 20px de altura

style
body {
background-size: cover;
}
style

Atribuindo o valor cover, o tamanho da imagem será ajustado de modo que ela seja sempre totalmente exibida na tela

style
body {
background-size: contain;
}
style

Já o valor contain redimensiona a imagem para que ela cubra o container, mesmo que para isso ocorram alguns eventuais cortes

BACKGROUND-ATTACHMENT

E, por último, podemos configurar a propriedade background-attachment atribuindo a ela os seguintes valores:

style
body {
background-attachment: scroll;
}
style

Essa é a configuração padrão, que faz com que a imagem de fundo role junto com o conteúdo

style
body {
background-attachment: fixed;
}
style

Esse outro valor, fixed, deixa a imagem fixada enquanto o conteúdo vai sendo rolado

SHORTHAND BACKGROUND

Agora que já vimos todas as propriedades, vamos aprender como juntar todas os valores dentro de uma propriedade

style
body {
background-color: black;
background-image: url(imagem.png);
background-position: center center;
background-repeat: no-repeat;
background-size: cover;
background-attachment: fixed;
}
style

Em vez de fazer todas essas declarações, podemos configurar todos os valores dentro da propriedade background:

style
body {
background: black url(imagem.png) center center no-repeat fixed;
background-size: cover;
}
style

OBS: Perceba que a shorthand background deveria conter a propriedade background-size, mas, como não funciona quando testamos no navegador, precisamos configurá-la separada das demais
