Estilos Inline

Entre as formas de colocar estilo em HTML, o estilo inline é o que deve ser mais evitado.

Ele consiste em adicionar estilo dentro das próprias tags

Exemplo:

<body style="background-color: blueviolet;">

Esse método polui muito a parte de HTML, além de mudar o estilo de somente uma tag específica,

tornando o trabalho muito cansativo quando se precisa alterar o estilo de várias tags.


Estilos Internos

É por isso que existe uma técnica melhor a se utilizar, que é definindo o estilo dentro da tag <style>: 

                    <style>
                    body {
                        background-color: blueviolet;
                    }
                    </style>

É recomendado que se use a tag após o título, obrigatoriamente na parte <head> do documento


Estilos Externos

Mas o melhor entre eles, principalmente se o site tiver mais de um documento, é o estilo externo.

Nele, o estilo tem um arquivo separado, que se conecta ao documento HTML através do link:

<link rel="stylesheet" href="style.css">

O parâmetro rel indica qual é o destino, que no caso é stylesheet, e o parâmetro href define o nome do documento, que

pode ser outro sem ser style.css // mas precisa conter .css
