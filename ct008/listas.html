LISTAS 📝

Ordered Lists:

Fazem uso da tag <ol>, que conté os <li> 
   <ol>
        <li> item 1
        <li> item 2
        <li> item 3
        <li> item 4
   </ol>
É possível escolher a forma de ordenação desta lista, usando o parâmetro type, que pode conter: 

        1 - 1, 2, 3, 4, 5
        A - A, B, C, D, E
        a - a, b, c, d, e
        I- I, II, III, IV, V
        i - i, ii, iii, iv, v

Unordered lists

Fazem uso da tag <ul>, mas diferente das ordered lists, não possuem qualquer indicação 
        <ul>
               <li> item 1
               <li> item 2
               ><li> item 3
               <li> item 4
        </ul>
É possível modificar a forma como esta lista aparecerá, usando o parâmentro type, que pode conter:
disc -
circle -
square -

Definition Lists

E, por último, temos as listas de definição, <dl>, nas quais temos as tags <dt> e <dd>

Exemplo:
DL
Definition List
DT
Definition term
DD
Definition Description
