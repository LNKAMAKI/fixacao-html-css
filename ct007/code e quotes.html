Trecho de códigos:

Já que há um amplo compartilhamento de códigos entre programadores,
existe a tag <code> para indicar que o texto se refere a um código
Exemplo:

function alert(){ window.alert('Exemplo de trecho de código') window.alert('Percebe que as letras ficam monoespaçadas?') }

Mas, como se pode notar, o código não ficou direitinho, isso porque é preciso
colocar a tag <pre> para que haja a formatação 


        
            function alert(){
            window.alert('Exemplo de trecho de código')
            window.alert('Percebe que as letras ficam monoespaçadas?')
            }

Citações

SIMPLES: <q>

Tell me and I forget, Teach me and I remember, Involve me and I learn

COMPLEXA: <blockquote>

Lorem ipsum dolor sit amet consectetur adipisicing elit. Id ullam atque earum eum provident aperiam pariatur? Eos odit quos ex, tempora magni, itaque voluptatum, repellat nulla iure nam architecto libero.
É possível dentro dessa tag colocar o parâmetro cite, onde é colocado um link para o texto original: 
              <blockquote cite="link do texto original">

Texto invertido

Caso precise por algum motivo do destino invertir o texto, use a tag <bdo> e defina o parâmetro dir: 
<bdo dir="rtl">
ou 
<bdo dir="ltr">

Abreviações:

E, por último, temos a tag <abbr>, de abreviação, que possui a seguinte estrutura: 
<abbr title="significado da abreviação">"abreviação"</abbr>
