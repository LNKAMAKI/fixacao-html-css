Tipografia

Tipografia é uma arte antiga que estuda técnicas de escrita para a apresentação de forma impressa.

Tipografia= týpos + graphía

Família Tipográfica

Família tipográfica é o conjunto de glifos - signos alfabéticos projetados para reprodução mecânica - que possuem as mesmas características anatômicas.

Fonte

Fontes são conjuntos de glifos que formam uma família tipográfica ou arquivos digitais que armazenam todos os formatos de glifos que compõe uma família tipográfica.

Fonte Serifada

Fontes serifadas são aquelas que possuem serifa - pequeno traço e prolongamento que ocorre no fim da haste de uma letra.

Fonte Não Serifada

As fontes não serifadas são aquelas que não possuem serifa.

Fonte Monoespaçada

Nesse tipo de fonte todas as letras ocupam o mesmo espaço, por isso recebem o nome de monoespaçadas.

Fonte Script

Fontes script ou handwriting são aquelas que simulam fontes escritas à mão .

Fonte Display

E, por último, existe as fontes display, que não se encaixam como nenhuma das anteriores.
