Gradiente em CSS

Gradiente é como se fosse um degradê, que pode ser tanto de uma ou mais cores. Para fazer esse efeito em CSS, usamos a propriedade background-image:

background-image: linear-gradient (to bottom,#E9D2F4,#73C2BE);

Linear-gradient indica que o tipo é gradiente linear.
To bottom indica a direção, ou seja, para baixo.
Os códigos indicam as cores do gradiente, em sequência.

O outro tipo de gradiente é o radial:

background-image: radial-gradient (circle,#E9D2F4,#73C2BE);

Como o próprio nome diz, ele gera um degradê em formato de círculo.

Além de to bottom, existe também: to top, to left e to right e também é possível especificar o grau, colocando o número seguido de deg.

Cajo haja algum problema ao usar essa propriedade, defina a altura para 100% nas configurações globais das CSS:

* {
          height:100%;
}
ou ajuste o background-attachment para que ele fique fixo:

background-attachment: fixed;
