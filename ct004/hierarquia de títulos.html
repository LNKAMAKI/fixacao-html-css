HTML → <h1>

Hierarquia de Títulos → <h2>

NÍVEIS:
  • <h1>(título principal)
  • <h2>(sub-título do título principal)
  • <h3>
  • <h4>
  • <h5>
  • <h6>
Exemplo:

Entendendo Portugal → h1

Introdução → h2

1. Justificação da Temática → h3

2. Caráter excecional → h3

3. Perguntas de Partida → h3

4. Metodologia de Investigação → h3

4.1 Etapa de investigação → h4

4.2 Instrumentos de análises → h4
